				二、使用配置要求及初始化流程
				（一）基础配置要求
				•操作系统：仅限Windows系统，要求Windows 10及以上版本（32位/64位均可，推荐64位，运行更流畅）；
				•Python环境：需提前安装Python 3.11及以上版本，确保环境变量配置正确（避免影响后续应用启动）；
				•浏览器要求：需安装Google Chrome浏览器（推荐）或Microsoft Edge浏览器，优先选择Google Chrome浏览器，兼容性更强、运行更稳定，可有效减少启动及使用过程中的异常问题。
				（二）初始化操作流程
				应用启动前需完成初始化设置，步骤如下，确保每一步操作准确无误，避免初始化失败：
				
				1.启动绿梵AI应用，首先进入初始化界面，界面将提示输入浏览器地址；
				以下以选择Google浏览器为例
				
				2.浏览器地址需填写对应浏览器（推荐Chrome）的应用程序安装路径，格式为完整本地文件路径，示例：C:\Program Files\Google\Chrome\Application\chrome.exe（可通过浏览器桌面图标“属性”中的“目标”获取准确路径）；

				（一）版本模块
				版本模块用于切换AI内核版本及设置检索模式，适配不同使用场景，具体包含：

			                                print(f"检测到Chrome版本: {version}")
			                                return version, path
			                    except Exception as e:
			                        print(f"检测版本失败 {path}: {e}")
			                        continue

			                "/Applications/Google Chrome.app/Contents/MacOS/Google Chrome"
